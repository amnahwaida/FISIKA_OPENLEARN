# Panduan Kontribusi

Terima kasih telah mempertimbangkan untuk berkontribusi di **FisikaSMA-OpenLearn**!

## Cara Berkontribusi
1. Fork repositori ini.
2. Buat branch baru (`git checkout -b fitur-baru`).
3. Lakukan perubahan pada materi (pastikan mengikuti standar Markdown dan penulisan matematika/LaTeX).
4. Commit perubahan Anda (`git commit -m 'Menambahkan materi/soal baru'`).
5. Push ke branch (`git push origin fitur-baru`).
6. Buat Pull Request baru.

## Aturan Penulisan
- Gunakan bahasa Indonesia yang baik, benar, dan mudah dipahami siswa SMA.
- Ikuti struktur template materi yang ada di PRD.
- Optimasi ukuran gambar (maksimal 200KB per gambar).
- Simpan gambar hanya di dalam folder `assets/[kelas]/`.
